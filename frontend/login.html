Application Distribuée de Gestion d’un Parc Automobile

Console sécurisée FleetOS

Accédez au cockpit IAM + Gateway pour piloter les services véhicules, conducteurs et incidents. Chaque connexion transite par http://localhost:8080 et aboutit sur un JWT signé contenant vos rôles métiers.

IAM (8090)

Contrôles d’identité, hashing BCrypt et SuperAdmin injecté via SUPERADMIN_*.

Gateway (8080)

Routes `/api/v1/**`, filtre JWT + rôles, forwarding Eureka.

Front (8081)

Nginx sert les écrans opérateurs/admin intégrés depuis le template responsive.

  • JWT multi-rôles
  • Spring Cloud Config
  • Observabilité Actuator
Rappel : les comptes administrateur sont créés par un SuperAdmin ou via le portail admin, jamais via l’inscription publique.

Centre d’identité IAM

Connexion opérateurs & administrateurs

Vos requêtes transitent par Gateway (8080) puis IAM vérifie l’email, le mot de passe et signe un JWT.

Identifiant requis : email professionnel déclaré dans Profiles Service.

Les mots de passe sont stockés en BCrypt (PasswordEncoder).
