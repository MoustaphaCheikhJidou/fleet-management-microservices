Application Distribuée de Gestion d’un Parc Automobile

Créer un compte opérateur (Carrier ou Driver)

Les inscriptions publiques n’autorisent que les rôles métiers terrain. Les administrateurs restent gérés depuis le portail backend pour éviter toute élévation de privilèges.

Les données sont stockées dans MySQL via IAM-service. Une confirmation apparaît ci-dessous.

L’auto-inscription est limitée à ces rôles métiers.

Important : Les comptes administrateur sont créés uniquement par un administrateur connecté via admin-dashboard.html.

Flux IAM

Chaque inscription alimente la plateforme FleetOS.

Les événements `iam.users.created` alimentent Profiles, Vehicles et Issues pour exposer immédiatement le nouveau membre dans les tableaux de bord. Le provisioning reste piloté par Eureka et Config Server.

  • Validation Spring Boot
  • JWT enrichi & diffusion RabbitMQ
  • Monitoring Actuator prêt
Astuce : lancez `docker compose up -d --build` avant de tester afin que Gateway, IAM, Vehicles et Issues soient tous enregistrés dans Eureka.
